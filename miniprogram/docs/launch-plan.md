# 🚀 冷启动方案:蹭 Chinamaxxing + Trump 访华

> 窗口期:2026 年 5 月 15 日起,热度预计持续 2-4 周(Chinamaxxing 趋势至少半年)

---

## 核心思路

```
不是等产品做完再推广,
是推广和开发同步进行:
先用内容养号 → 小程序上线后用内容导流。
```

## 第一周:小红书/抖音内容先行(小程序还在审核)

### 发什么

**形式:图文 / 短视频,每天 2-3 条**

| # | 选题 | 形式 | 蹭的热点 |
|---|------|------|---------|
| 1 | 「中国人含量测试预告」 | 图文,放 3-4 道题预览 | Chinamaxxing |
| 2 | 「Musk 在北京的中国人含量」 | AI 生成 Musk 泡脚/喝枸杞图 | Trump 访华 |

## 第二周:小程序上线 + 导流

### 上线当天动作清单

```
□ 小红书发"终于做出来了"的内容 + 小程序名称引导搜索
□ 朋友圈发自己测试的海报(先发 3-5 个不同结果)
□ 发到 3-5 个微信群(大学生群/打工人群/互联网群)
□ 抖音发"测试过程录屏"短视频
□ 让 5-10 个朋友测完分享海报到朋友圈
```

### 裂变机制(已内置)

```
1. 结果页自带分享按钮(微信好友 + 朋友圈)
2. 海报带小程序码(朋友圈裂变核心)
3. 结果文案本身就是社交货币:
   - "我的中国人含量 97%!!!"
   - "连你妈都觉得你不像中国人"
   - 这种文案用户会主动截图发
```


## 总结

这个产品的核心不是"做得多精美",而是:
1. **快**:蹭热点窗口期
2. **准**:踩中 Chinamaxxing 的共鸣点
3. **分享驱动**:结果图值得发朋友圈

做到这三点,技术再糙都能跑起来。
